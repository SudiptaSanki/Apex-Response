# More Product Ideas

This file collects additional ideas that forks can explore beyond the current MVP.

## Guest Experience

- Family reunification status board
- Accessibility mode with mobility-safe routes
- Low-vision mode with high-contrast step cards
- Audio guidance mode for evacuation or shelter instructions
- Room-device alert handoff to hotel TVs or tablets
- Offline emergency guide cache for poor connectivity

## Staff and Management

- Floor warden task assignment board
- Incident timeline with manual notes and approvals
- Staff panic button and welfare check queue
- Shift-aware staffing readiness indicator
- Generator, backup power, and utility status panel
- Post-incident after-action report generator

## First Responders

- Tactical ingress and egress overlays
- Hazard zone annotations by floor
- Utility shutoff checklist
- Stairwell clearance board
- Hazmat inventory pack
- Incident handoff export for dispatch records

## Data and Intelligence

- OpenFEMA disaster context
- Air quality and wildfire smoke feeds
- Flood risk overlays
- Local dispatch station proximity cards
- Indoor occupancy confidence scoring
- Predictive crowding risk by zone

## Open Source Growth Ideas

- Component library extraction
- Storybook or visual docs
- GitHub Actions for build verification
- API adapters folder for multiple providers
- Localization support
- Theming support for different organizations
